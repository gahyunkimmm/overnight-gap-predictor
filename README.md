# 야간 신호 기반 한국 반도체주 개장 갭 예측

어젯밤 미국 시장(반도체) 신호로 **다음 한국 거래일의 개장 갭(시가 등락률)**을 추정한다.
삼성전자·SK하이닉스는 미국 정식 ADR이 없으므로, 직접 가격 대신 **상관 자산**을 신호로 사용한다.

## 아이디어

미국 장(D-1 밤)은 한국 장(D일 아침)보다 먼저 끝난다.
→ 그날 밤 글로벌 반도체 업황이 다음 날 한국 개장 갭에 얼마나 반영되는지를 회귀로 추정.

| 신호 | 티커 | 의미 |
|------|------|------|
| 마이크론 | `MU` | 메모리 직접 경쟁사 (삼성·하이닉스 최강 선행지표) |
| 필라델피아 반도체지수 | `^SOX` | 반도체 섹터 전반 |
| 한국 ETF | `EWY` | 한국 증시 전반 |
| 원/달러 | `KRW=X` | 환율(이 종목/기간엔 영향 미미) |

## 검증 결과 (2023-01 ~ 2026-06, 810거래일)

| 지표 | 삼성전자 | SK하이닉스 |
|------|---------|-----------|
| out-of-sample R² | 0.32 | 0.45 |
| 방향 적중률 | 72.0% | 74.5% |
| (단순 한쪽찍기 기준선) | 58.4% | 51.9% |

단변량 상관: SOX +0.58/+0.67, MU +0.56/+0.63, 환율 +0.09/+0.06 (환율은 사실상 무의미).

## 사용법

```bash
pip install -r requirements.txt

# 상관관계·예측력 검증 리포트
python overnight_gap_verify.py

# 매일 아침(한국 장 시작 전) 실행 — 오늘 갭 추정
python overnight_gap_predict.py
```

`overnight_gap_predict.py`는 예측을 `predictions_log.csv`에 누적 저장하므로,
실제 개장가와 비교해 사후 정확도를 추적할 수 있다.

## 매일 자동 실행 (GitHub Actions)

`.github/workflows/daily-predict.yml`이 **한국 평일 아침(07:00 KST)**에 자동 실행되어
그날 예측을 `predictions_log.csv`에 누적 커밋한다. 로컬 PC를 켜둘 필요가 없다.

- 수동 실행: GitHub 저장소 → **Actions** 탭 → *일일 개장 갭 예측* → **Run workflow**
- 스케줄은 `cron: "0 22 * * 0-4"` (UTC) = 한국 월~금 07:00

> 대시보드(Streamlit)는 접속 시마다 실시간 계산되므로 갱신용 자동 실행이 필요 없다.
> Actions의 목적은 **일일 예측을 기록으로 남겨 사후 정확도를 추적**하는 것이다.

## ⚠️ 한계 (중요)

- 예측 대상은 **개장 갭**이며, 갭은 **개장가 그 자체에 이미 반영되어 열린다.**
  예측이 맞아도 일반 투자자는 갭 수익을 그대로 취할 수 없다.
- 장중(개장→종가)은 예측력이 급감한다(거의 랜덤).
- 최근 반도체 슈퍼사이클 데이터에 적합되어 있어, 업황 체제 변화 시 상관관계가 깨질 수 있다.
- **투자 권유가 아니며**, 갭 방향·강도를 가늠하는 참고용 분석 도구다.
