# 야간 신호 기반 한국 반도체·대형주 개장 갭 예측

어젯밤 미국 시장 신호로 **다음 한국 거래일의 개장 갭(시가 등락률)**을 추정한다.
삼성전자·SK하이닉스·현대차는 미국 정식 ADR이 없으므로, 직접 가격 대신 **상관 자산**을 신호로 사용한다.

## 아이디어

미국 장(D-1 밤)은 한국 장(D일 아침)보다 먼저 끝난다.
→ 그날 밤 글로벌 반도체·시장 흐름이 다음 날 한국 개장 갭에 얼마나 반영되는지를 회귀로 추정.

## 모델

- **입력 신호 9종** (전부 yfinance로 안정 수집)

  | 신호 | 티커 | 신호 | 티커 |
  |------|------|------|------|
  | 마이크론 | `MU` | 엔비디아 | `NVDA` |
  | 필라델피아 반도체지수 | `^SOX` | TSMC | `TSM` |
  | 한국 ETF | `EWY` | 나스닥 종합 | `^IXIC` |
  | 원/달러 | `KRW=X` | 변동성지수 | `^VIX` |
  | | | S&P 500 | `^GSPC` |

- **방법**: 표준화 + **릿지 회귀(λ=10)**. 반도체 신호들이 서로 강상관(다중공선성)이라
  릿지로 안정화. 신호 구성·정규화 선택 근거는 [`experiment.py`](experiment.py) 참고.

## 검증 결과 (워크포워드, 2022-01 ~ 2026-06)

70/30 단일 분할이 아니라 **확장창 워크포워드**(과거 전부로 학습 → 다음날 예측을 반복)로 검증한다.
실전 운용과 동일한 방식이라 더 정직한 수치다.

| 지표 (개장 갭) | 삼성전자 | SK하이닉스 | 삼성전기 | 현대차 |
|------|---------|-----------|---------|--------|
| 워크포워드 R² | 0.42 | 0.51 | 0.31 | 0.26 |
| 방향 적중률 | 73% | 75% | 70% | 68% |

### 핵심: 갭은 맞지만, 그 너머는 못 맞춘다

같은 신호로 **장중(개장→종가)** 수익을 예측하면:

| 지표 (장중) | 삼성전자 | SK하이닉스 | 삼성전기 | 현대차 |
|------|---------|-----------|---------|--------|
| 방향 적중률 | 47% | 51% | 53% | 47% |
| R² | ≈ 0 (음수) | ≈ 0 | ≈ 0 | ≈ 0 |

→ 야간 신호는 **개장 갭은 잘 설명하지만 장중 움직임엔 예측력이 사실상 0**이다.
갭은 개장가에 이미 반영되어 열리므로, 이 모델은 **수익 도구가 아니라 방향 가늠용**임이
데이터로 확인된다(대시보드 "갭 vs 장중 예측력" 패널에서 매번 갱신).

## 사용법

```bash
pip install -r requirements.txt

# 상관관계·예측력 검증 리포트
python overnight_gap_verify.py

# 매일 아침(한국 장 시작 전) 실행 — 오늘 갭 추정
python overnight_gap_predict.py

# 단위 테스트 (네트워크 불필요)
python test_model.py
```

> 견고성: 미국 신호 일부가 수집 실패해도 **가용 신호로 자동 축소**해 동작하며,
> 신호가 너무 적으면 명확한 에러를 낸다(앱이 죽지 않음).

`overnight_gap_predict.py`는 예측을 `predictions_log.csv`에 누적 저장하므로,
실제 개장가와 비교해 사후 정확도를 추적할 수 있다.

### 웹 대시보드

```bash
streamlit run app.py
```

신호 요약 · 종목별 갭 예측 · 최근 30일 적합도 차트에 더해, 누적 예측이 쌓이면
**실측 정확도 패널**(예측 vs 실제 개장 갭, 적중률·MAE)을 자동 표시한다.

## 유사 서비스와의 차이

- [코스피랩](https://kospilab.com) 등은 해외 거래가를 환산해 보여주는 **실시간 미러**(지금 얼마)다.
- 본 프로젝트는 상관 신호로 **다음날 갭을 예측**하는 **검증된 회귀 모델**(내일 어디로)이며,
  방법론을 공개하고 **백테스트·실측 정확도로 예측력을 입증**한다는 점이 다르다.

## 매일 자동 실행 (GitHub Actions)

`.github/workflows/daily-predict.yml`이 **한국 평일 아침(07:00 KST)**에 자동 실행되어
그날 예측을 `predictions_log.csv`에 누적 커밋한다. 로컬 PC를 켜둘 필요가 없다.

- 수동 실행: GitHub 저장소 → **Actions** 탭 → *일일 개장 갭 예측* → **Run workflow**
- 스케줄은 `cron: "0 22 * * 0-4"` (UTC) = 한국 월~금 07:00

> 대시보드(Streamlit)는 접속 시마다 실시간 계산되므로 갱신용 자동 실행이 필요 없다.
> Actions의 목적은 **일일 예측을 기록으로 남겨 사후 정확도를 추적**하는 것이다.

## ⚠️ 한계 (중요)

- 예측 대상은 **개장 갭**이며, 갭은 **개장가 그 자체에 이미 반영되어 열린다.**
  예측이 맞아도 일반 투자자는 갭 수익을 그대로 취할 수 없다.
- 장중(개장→종가)은 예측력이 급감한다(거의 랜덤).
- 최근 반도체 슈퍼사이클 데이터에 적합되어 있어, 업황 체제 변화 시 상관관계가 깨질 수 있다.
- **투자 권유가 아니며**, 갭 방향·강도를 가늠하는 참고용 분석 도구다.
